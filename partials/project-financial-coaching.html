Description

Designed and built an agent-based financial coaching and expense intelligence platform using LangGraph. A router + React agent pattern selects the correct agent per task.

The system processes invoices using Vision-Language Models (VLMs) and Docling to extract structured information from images and PDFs. Agents handle parsing, validation, cleaning, and reasoning about spending behavior.

The platform runs serverless on AWS: S3 stores raw documents, Lambda executes processing, and SQS supports asynchronous workloads. Data is stored in MongoDB Atlas, and Atlas Vector Search enables semantic retrieval to support personalized coaching insights.

Tech stack: Adaptive RAG, LangGraph, LangChain, Vision-Language Models, Docling, AWS Lambda, SQS, S3, MongoDB Atlas, Atlas Vector Search, FastAPI
