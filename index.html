Financial Coaching through Document Intelligence
Developed an agent-driven financial coaching and expense intelligence system powered by LangGraph and LLM reasoning. The solution uses a multi-agent workflow to parse invoices, extract semistructured financial information, and reason about spending patterns rather than simply classify text. The pipeline uses Docling + LLMs for document understanding, and LangChain for semantic enrichment and category inference. Each invoice is processed into item-level structured data, where agents collaborate to validate, categorize, and provide actionable financial insights. The platform integrates with AWS S3 for document storage, deploys containerized microservices through AWS ECR, and stores enriched records in MongoDB Atlas, enabling real-time analytics and financial behavior coaching.
AI4Good: Detecting Anomalies in Energy Consumption using Machine Learning
Designed a machine learning solution to detect abnormal consumption patterns in energy usage data, supporting utility companies in identifying inefficiencies, technical issues, and potential non-technical losses. The project focused on responsible AI and sustainability by aligning data-driven insights with operational impact and energy efficiency goals. Applied segmentation and unsupervised learning techniques including K-Means, Isolation Forest, Local Outlier Factor, and One-Class SVM. The workflow incorporated ETL processing, feature engineering (cyclical encoding and power factor computation), PCA-based dimensional analysis, and cluster interpretation for actionable insights. The final deliverables included a production-ready anomaly scoring datamart, dashboard visualizations for decision-making, and a scalable analytics approach built using Dask, scikit-learn, and Gradio to enable AI-assisted intervention planning.
